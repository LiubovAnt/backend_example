# Панель администратора

- manage.py             - Консольная утилита для управления созданным проектом. В этом и следующих уроках вам часто придётся ей пользоваться.
- config/                - Конфигурация приложения:
|--> componenents/           - Подключаемые компоненты:
    |--> database.py            - Информация для подключения к PostgreSQL базе данных
|--> settings.py            - Файл настроек проекта
|--> urls.py                - Содержит настройки путей (url), которые будут задействованы в проекте
|--> asgi.py                - Файлы, которые понадобятся позже для запуска проекта на боевом сервере 
|--> wsgi.py                - Файлы, которые понадобятся позже для запуска проекта на боевом сервере 
- movies                - Приложение:
|--> migrations/            - Папка для миграций базы данных:
    |-->0001_initial.py         - Начальная миграция. Создание таблиц с жанрами, дильмами и персоами и связей между ними
    |-->0002_filmwork_certificate_filmwork_file_path_and_more.py
|--> admin.py               - Cоздаёт административные интерфейсы для моделей приложения
|--> apps.py                - Определяет настройки приложения 
|--> models.py              - Xранит определение моделей приложения
|--> #tests.py              - Xранит тесты приложения (нет в данной реализации)
|--> #views.py              - Определяет обработчики, которые получают запросы, обрабатывают их и возвращают ответ (нет в данной реализации)

python manage.py createsuperuser - создание админа
Для запуска миграций:
- пропускаем 0001_initial, т.к. база уже существует:
     python manage.py migrate --fake movies 0001
- вносим вторую миграцию
    python manage.py migrate movies 0002
Компиляция перевода:
- python manage.py compilemessages -l en -l ru
